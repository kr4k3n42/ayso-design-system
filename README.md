# AYSO Design System

The canonical source of truth for AYSO brand design — for human designers, AI tools, and outside vendors.

## What's in this repo

- **`DESIGN.md`** — The master AYSO Brand Design System. Logo, color, typography, voice, photography, apparel, regional identity, and the Lock/Canvas/Remix system.
- **`PRODUCT-DESIGN.md`** *(coming soon)* — The companion product design system governing the MyAYSO app, registration software, and digital tools.

## Who this is for

- **AYSO national brand team** — maintainers and reviewers
- **Regional commissioners** — reference for regional brand expression
- **Outside design firms and agencies** — the brand bible
- **AI design tools** (Claude Design, Stitch, v0, Cursor) — connect this repo to keep output on-brand
- **The future AYSO AI brand configurator** — reads from this repo

## How to use with AI tools

### Claude Design
Connect this repo during project setup. Claude Design will read DESIGN.md and apply AYSO tokens, type, and rules to every generation in the project.

### Other AI tools
Copy the contents of DESIGN.md into the project context or system prompt.
